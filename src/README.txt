
Sistema de Gestión de Inventarios - Proyecto POO 2

Este proyecto demuestra:
- Clase abstracta (Producto)
- Herencia (ProductoElectronico, ProductoPerecedero)
- Interfaz (IAlmacenable)
- Colecciones (ArrayList)
- Manejo de excepciones (try-catch)
- Persistencia de datos en archivo .txt

Cómo ejecutar:
1. Abrir el proyecto en NetBeans, IntelliJ o VS Code.
2. Ejecutar Main.java
3. Usar el menú del sistema.

Para GitHub:
Subir toda la carpeta del proyecto a un repositorio y copiar el enlace en el PDF.
