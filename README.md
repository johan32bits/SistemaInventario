## 🚀 Características del Proyecto
 Este sistema aplica los conceptos avanzados de POO solicitados en la guía docente 
*  **Herencia y Polimorfismo:** Implementación de subclases `ProductoElectronico` y `ProductoPerecedero` que extienden de la superclase `Producto` 
* **Abstracción:** Uso de una **Clase Abstracta** (`Producto`) y una **Interfaz** (`IAlmacenable`) 
 **Colecciones:** Gestión dinámica de datos mediante `ArrayList<Producto>` 
 ]**Manejo de Excepciones:** Uso de bloques `try-catch` para validación de entradas y lectura de archivos .
 **Persistencia:** Guardado de datos en archivos planos (.txt) para evitar la pérdida de información
