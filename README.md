## 🚀 Características del Proyecto
[cite_start]Este sistema aplica los conceptos avanzados de POO solicitados en la guía docente[cite: 9, 12]:

* [cite_start]**Herencia y Polimorfismo:** Implementación de subclases `ProductoElectronico` y `ProductoPerecedero` que extienden de la superclase `Producto`[cite: 16, 47].
* [cite_start]**Abstracción:** Uso de una **Clase Abstracta** (`Producto`) y una **Interfaz** (`IAlmacenable`)[cite: 17, 47].
* [cite_start]**Colecciones:** Gestión dinámica de datos mediante `ArrayList<Producto>`[cite: 18, 47].
* [cite_start]**Manejo de Excepciones:** Uso de bloques `try-catch` para validación de entradas y lectura de archivos[cite: 19].
* [cite_start]**Persistencia:** Guardado de datos en archivos planos (.txt) para evitar la pérdida de información[cite: 20].
